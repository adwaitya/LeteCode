<h3>345. Reverse Vowels of a String</h3>
<div>
<p>
Given a string s, reverse only all the vowels in the string and return it.</p>
<p>
The vowels are 'a', 'e', 'i', 'o', and 'u', and they can appear in both cases.
</p>
<pre>
Example 1:

Input: s = "hello"
Output: "holle"
Example 2:

Input: s = "leetcode"
Output: "leotcede"
</pre>
